# The List Of Tools You Need To Get Started

## 0- Linux
We will use Linux as our operating system every other tool will run on it.
Linux has many distributions but we will be using Ubuntu.

## We can divide our tools into 3 main categories
1. Automation
2. Monitoring/ELK
3. Project Management

## 1- Docker (Containerization)
The number one tool for DevOps is Docker we use it for conainerization.
Means we put our code and environment inside docker container and anyone can use that container, it will be in a virtual environment separate from our machine.

## 2- Jenkins (CI/CD)
We will use Jenkins for CI/CD piplelines and automation.

## 3- Git (Source Code Management)
We will use it for version control and source code management.

## 4- Terrafrom
It is an IAC (Infrastruture As Code) tool, it helps in cloud or on prem resources.

## 5- Ansible
It is a configuration tool

## 6- Kubernetes
It is a container orchestration platform

## -7 Prometheus (Monitoring)
We use it for event monitoring and alerting

## 8- ELK Stack
Elasticsearch, Logstash, Kibana

## 9- Jira/Trello (Project Management)

## 10- Any Cloud Platform (AWS, Azure, GCP)
