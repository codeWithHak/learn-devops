# The List Of Tools You Need To Get Started

## 0- Linux
We will use Linux as our operating system every other tool will run on it.
Linux has many distributions but we will be using Ubuntu.

## We can divide our tools into 3 main categories
1. Automation
2. Monitoring/ELK
3. Project Management

## 1- Docker
The number one tool for DevOps is Docker we use it for conainerization.
Means we put our code and environment inside docker container and anyone can use that container, it will be in a virtual environment separate from our machine.
