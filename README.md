# 60 Days DevOps Learning Journey

This repo documents my 60-day DevOps journey from ground zero to production-ready, covering Linux fundamentals to Kubernetes orchestration on Ubuntu.

## Timeline
**Start Date:** November 12, 2025  
**Target End Date:** January 10, 2026  
**Current Progress:** Day 0/60

## Goal
Master core DevOps tools and practices to become a proficient DevOps Engineer through consistent hands-on learning.

## Course Following
[Complete DevOps Roadmap 2025 by DevOps Molvi](https://youtu.be/m1zOhVum4oM?si=u1TH2nScl9VKAa_M)

## Learning Roadmap

### Phase 1: Linux Foundation (Days 1-12)
- Linux basics and file system
- SSH and key authentication
- Essential Linux commands

### Phase 2: Containerization with Docker (Days 13-27)
- Docker fundamentals and architecture
- Dockerfiles and multi-stage builds
- Docker volumes, networking, and compose
- Docker Hub and container registries

### Phase 3: Version Control with Git (Days 28-35)
- Git fundamentals and workflow
- Branching and collaboration
- GitHub essentials

### Phase 4: CI/CD Pipelines (Days 36-42)
- CI/CD concepts and best practices
- GitHub Actions
- GitLab CI/CD pipelines

### Phase 5: Infrastructure as Code (Days 43-50)
- Terraform fundamentals
- Cloud deployments (Azure, GCP)
- State management and modules

### Phase 6: Container Orchestration (Days 51-60)
- Kubernetes architecture and concepts
- kubectl commands and cluster management
- Deployments, Services, and RBAC
- Production-ready configurations

## Environment
- **OS:** Ubuntu Linux
- **Learning Style:** Hands-on practice with real-world scenarios
- **Documentation:** Daily progress notes and code examples

## Note
This is a self-paced learning journey with a 60-day target, not a strict challenge. The goal is consistent progress and deep understanding.
