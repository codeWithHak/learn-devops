# Learn DevOps

This repo documents my DevOps learning journey from ground zero to production-ready skills, covering Linux fundamentals to Kubernetes orchestration on Ubuntu.

## Timeline
**Course Start Date:** November 12, 2025  
**Initial Course Target:** 60 days  
**Learning Approach:** Lifelong continuous learning

## Goal
Master core DevOps tools and practices to become a proficient DevOps Engineer through consistent hands-on learning. This is a continuous journey where I'll complete the foundational course in 60 days and keep expanding my knowledge beyond.

## Course Following

## Learning Roadmap

### Phase 1: Linux Foundation
- Linux basics and file system
- SSH and key authentication
- Essential Linux commands

### Phase 2: Containerization with Docker
- Docker fundamentals and architecture
- Dockerfiles and multi-stage builds
- Docker volumes, networking, and compose
- Docker Hub and container registries

### Phase 3: Version Control with Git
- Git fundamentals and workflow
- Branching and collaboration
- GitHub essentials

### Phase 4: CI/CD Pipelines
- CI/CD concepts and best practices
- GitHub Actions
- GitLab CI/CD pipelines

### Phase 5: Infrastructure as Code
- Terraform fundamentals
- Cloud deployments (Azure, GCP)
- State management and modules

### Phase 6: Container Orchestration
- Kubernetes architecture and concepts
- kubectl commands and cluster management
- Deployments, Services, and RBAC
- Production-ready configurations

## Environment
- **OS:** Ubuntu Linux
- **Learning Style:** Hands-on practice with real-world scenarios
- **Documentation:** Progress notes and code examples

## Note
This is a continuous learning journey. While I aim to complete the foundational course in 60 days, DevOps is an ever-evolving field, and learning never stops.
